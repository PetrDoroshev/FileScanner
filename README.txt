Requirements:
    1. OS Windows 10/11
    2. Cmake 3.15+
    3. GCC ≥ 11, Clang ≥ 14, MSVC ≥ Visual Studio 2019 16.11


How to build and run:
    1. cd FileScanner
    2. mkdir build
    3. cd build
    4. cmake ..
    5. cmake --build .
    6. cmake --install . --prefix ./
    7. ./bin/FileScanner-CLI.exe
